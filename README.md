## Overview

Hackathon Hub provides a centralized platform where users can:
- Explore hackathon domains
- View upcoming events
- Practice coding skills
- Check leaderboards
- Receive notifications
- Submit feedback

The project currently focuses on frontend functionality with PHP used for structure and future backend expansion.

---

## Features

- User Login and Sign Up (UI-based)
- Hackathon Domains Listing
- Events Information Page
- Practice Section for Skill Improvement
- Leaderboard Display
- Notifications Page
- Feedback Submission Form
- Common CSS styling across all pages
- Clean and simple user interface

---

## Technology Stack

- **Frontend:** HTML5, CSS3  
- **Backend:** PHP  
- **Server:** Apache  
- **Local Server Tools:** XAMPP / WAMP
